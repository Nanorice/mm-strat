## 💡 Context/Memory

### The root cause the user identified (most important takeaway)

**Row-grain training vs event-grain backtest is the structural mismatch.**
`t3_sepa_features` holds ~3.4M rows because every ticker contributes ~250 rows
per year. The dominant features (`low_52w` median 0%/day, `adr_20d` median
1.6%/day) are nearly constant across consecutive rows for the same ticker, so
the model fits hundreds of near-duplicate rows as independent samples. The
label `y_homerun` is identical for all 20 consecutive day-rows that share the
same forward 20-day window. Net effect: high-ADR names dominate gradient mass
as both features and labels, even though they represent only a handful of
distinct economic events.

This single issue explains all four anomalies:
- `adr_20d` SHAP is 5× any other feature (concentration of label-mass).
- 2021 had only 11 trades (no ADR spike → no eligible tickers).
- Walk-forward AUC 0.78–0.79 looks great (per-row metric on millions of rows)
  while backtest is regime-specific (per-event metric on dozens of episodes).
- OOS backtest selects 7 names (LUMN/GOSS/ESPR/APYX/OPEN/AXTI/IBRX) — the same
  high-ADR episodes the model trained to find.

### Strategy ambiguity (the second part of the same issue)

The cookbook's `consec=3 → buy → no re-entry while open → re-enter after slot
frees` was an arbitrary choice that interacted badly with feature staleness:
the same name keeps qualifying because its score barely moves, so the
slot-rotation amplifies the duplicated-ticker training bias.

### What this means for the model

- m01_rank's signal is real (AUC 0.79 OOS, beats prod across all metrics
  except trade count parity).
- The 28× total return / 44% per-trade / 13.6 Sharpe is a **regime-amplified
  upper envelope**, not a tradeable expectation.
- The deployable edge is somewhere between prod's 0.8% per trade and
  m01_rank's 44% — closer to prod's number once sample bias and regime
  concentration are addressed.
- The model is **regime-specific** (volatility-spike continuation in
  near-52-week-low setups), not a continuous edge. 2021 calm bull → 11
  trades, 2020/2022/2023 turbulence → 60–95 trades each. Deployment must
  account for this — likely as one of several specialised models gated by
  M03 regime, not as a primary signal.
