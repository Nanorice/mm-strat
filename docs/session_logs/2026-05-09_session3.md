### Vectorized vs Sequential Discovery
Ran parameter sweep with vectorized backtest, then validated best config with sequential:

| Metric | Vectorized (fake) | Sequential (real) |
|--------|-------------------|-------------------|
| Trades | ~500+ | 118 |
| Sharpe | 1.56 | 0.55 |
| Total Return | 509% | 55% |
| Win Rate | 56% | 37% |
| Max Drawdown | — | 35.7% |

**Root cause**: Vectorized backtest has unlimited concurrent positions and infinite capital. It enters ALL qualifying tickers across time, not constrained by `max_positions` slots.

### Sequential Backtest Features
- Day-by-day loop with real capital tracking
- `max_positions` enforced (slot management)
- Re-entry allowed after `cooldown_days`
- Position sizing based on current cash
- Mark-to-market daily equity curve
- Entry/exit costs deducted realistically
