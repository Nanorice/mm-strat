# Session Logs — Index

Chronological record of research sprints. Each sprint's **`README.md` is its front page** —
start there. Older sprints predate the folder convention and keep flat layouts.

## Layout convention (sprint_13 onward)
```
sprint_N/
  README.md      <- front page: dates, status, headline outcomes, folder map, roadmap
  logs/          <- dated session handovers (YYYY-MM-DD.md), one per work session
  plans/         <- forward-looking design/plan docs (written before the work)
  verdicts/      <- findings, reports, issues, playbooks (what we concluded)
  cells/         <- notebook-cell artifacts (no-direct-.ipynb-edit rule); scratch once applied
```
Caches, parquets, `.log` output → **`data/`**, never `docs/`. Cross-session facts → **memory**, not here.
