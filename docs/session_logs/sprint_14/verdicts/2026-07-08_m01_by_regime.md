**Every trunk has NEGATIVE separation** — bull days precede LOWER fwd return because the rebound

### (d) the two stat tests DISAGREE — the honest lesson
- **Bootstrap CI (day-resampled):** gap excludes 0 on 25y → REAL. On the 3-year smoke it STRADDLED 0
  → not established. Sample size / regime coverage matters; the smoke was crash-heavy and thin.
- **Kruskal-Wallis:** H=11,814, p≈0 — but at 9M rows with autocorrelated days, iid tests call any
  difference significant. p is meaningless here. **The block-bootstrap is the read to trust.**
  Keeping both was the right call — the p-value alone would have over-claimed.
